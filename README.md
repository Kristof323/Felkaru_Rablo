# 🦾 Felkaru Rabló

C# alapú konzolos projekt, iskolai feladat.

## Leírás

Egy egyszerű karakteralapú program C# nyelven, amely egy karakter (felkarú rabló) viselkedését szimulálja. Objektumorientált elvek alapján készült.

## Technológiák

- **Nyelv:** C#
- **.NET** keretrendszer

## Használat

1. Klónozd a repót:
   ```bash
   git clone https://github.com/Kristof323/Felkaru_Rablo.git
   ```
2. Nyisd meg Visual Studio-ban
3. Fordítsd és futtasd a projektet
